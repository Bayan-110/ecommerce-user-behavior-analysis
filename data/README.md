# 数据来源

本项目使用 **Synerise 公开数据集**（2025年发布）。

## 下载方式

访问 https://recsys.synerise.com/data-set 下载 `Synerise_dataset.zip`（约1.9GB）

## 数据说明

| 文件 | 记录数 | 说明 |
|------|--------|------|
| page_visit.parquet | 199,451,980 | 页面访问记录 |
| add_to_cart.parquet | 7,541,117 | 加购记录 |
| product_buy.parquet | 2,318,502 | 购买记录 |
| search_query.parquet | 13,223,769 | 搜索记录 |
| remove_from_cart.parquet | 2,688,894 | 移出购物车记录 |
| product_properties.parquet | - | 商品属性（含品类、价格） |

## 注意

原始数据文件较大，未上传至 GitHub。请自行下载后放在此目录下。
